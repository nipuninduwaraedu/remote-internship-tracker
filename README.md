# 🚀 Remote Internship Tracker

A modern web application built with **Next.js**, **TypeScript**, and **Tailwind CSS** that allows users to browse the latest remote internships using the Remote OK API.

This project was developed as a portfolio project to demonstrate frontend development skills for Web Development internship applications.

## ✨ Features

- Browse the latest remote internships
- Search internships by keyword
- Popular technology tags
- Dynamic internship details page
- Save internships using Local Storage
- Responsive design
- Company information
- Technology tags
- Salary information (when available)
- Posted date
- Direct Apply button
- Professional UI built with Tailwind CSS

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/remote-internship-tracker.git
```

### Navigate to the project

```bash
cd remote-internship-tracker
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
